HOW TO GET LESS LAG - written by snipergaming888

1. Unistall unused extenstions and apps, if possible.

2. If you can, turn on GPU rasterization.

chrome://flags/#enable-gpu-rasterization

3. if you can, in the drop down box, select Enables Hyper thread.(WARNING, ENABLING THIS WILL MAKE YOU MORE VULRNERABLE TO MEMORY EXPLOITS ON WEBPAGES, BE CAREFULL!)

chrome://flags/#scheduler-configuration

4. Clear your cache every so often.
